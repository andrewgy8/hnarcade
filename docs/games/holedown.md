# Holedown

A mesmerizing ball-shooting puzzle game about digging deep underground through layers of blocks.

## About

Holedown is a satisfying arcade puzzle game where you shoot balls downward to break blocks and dig deeper underground. Each planet presents new challenges with different block layouts and mechanics. The game features a simple one-touch control scheme that's easy to pick up but hard to master.
