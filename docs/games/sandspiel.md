A falling sand game built with Rust and WebAssembly. Draw with elements and watch them interact.

## About

Sandspiel is a pixel physics simulation sandbox. Pick from elements like sand, water, fire, oil, plants, and more, then paint them onto the canvas and watch the simulation play out.
